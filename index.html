Brief History of Git

Created in 2005 by Linus Torvalds

Why?

Working on Linux kernel and needed better version control system than what they were using (BitKeeper). So, if it doesn't exist (and you're Linus Torvalds), you create it yourself.

Had been using tarballs and patches so he built the system around a similar concept: snapshots of a point in time stored as binary objects in directory.

Finished the code in less than a month and 1.0 was released later that year (only on version 1.7 in 2012)

Purposes

Need to be able to incrementally advance
(and roll back) projects

Good ol' Copy & Paste

Subversion, SourceSafe, TFS

 

Need to work with a team

Every person has full copy of repository

Mercurial (Hg) is similar to Git

GitHub

So, Git is cool! But, what is GitHub?

  • Hosted git repositories for everyone!
  • Original Slogan: Git hosting: no longer a pain in the ass
  • Free for open source, affordable for closed projects
  • Founded in 2008. As of February 2012, 1.3+ million users and 2+ million repositories
  • 3 founders in 2008. 8 employees in 2010. 70+ employees right now. NO venture capital.
  • Gists, Project Pages, Issues, GitHub for Mac, GitHub for Windows, Watching Repo's, and more...
